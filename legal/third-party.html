제3자 서비스 고지

XENA Universe는 정적 웹사이트와 게임 저장, 로그인, 보안 검증을 위해 외부 인프라 서비스를 사용합니다.

시행일: 2026-07-19

1. 사용 서비스

서비스목적처리될 수 있는 정보
Google Firebase AuthenticationGoogle 로그인, 계정 식별UID, 이메일 일부, 표시 이름, 프로필 이미지 URL
Google Cloud Firestore게임 저장, 보유 항목, 재화 원장, 온라인 대전 데이터게임 진행 데이터, 닉네임, 재화, 보상 기록
Cloud Functions for Firebase보상 지급, 결제 검증, 부정 이용 방지계정 ID, 보상/결제 이벤트 ID, 결과값
GitHub Pages정적 웹사이트 호스팅일반적인 접속 로그가 처리될 수 있음

2. 국외 처리 가능성

Google 및 GitHub 인프라는 글로벌 클라우드 서비스이므로 데이터가 국외 리전에 저장 또는 처리될 수 있습니다. 운영자는 필요한 최소 정보만 저장하고, 서비스 보안을 위해 접근 권한을 제한합니다.

3. 결제 서비스 도입 시

향후 Stripe, Toss Payments, 앱마켓 결제 등 결제대행사를 도입할 경우 결제 승인, 취소, 환불 처리를 위해 결제대행사가 결제 정보를 처리합니다. XENA Universe는 카드번호 전체 등 민감 결제정보를 직접 저장하지 않는 구조를 원칙으로 합니다.

4. 외부 링크

사이트에는 YouTube, Wattpad, Patreon, Spotify, Marpple 등 외부 링크가 포함될 수 있습니다. 외부 사이트 이용에는 각 서비스의 약관과 개인정보처리방침이 적용됩니다.

5. 보안 설계

  • 클라이언트가 지갑 데이터를 직접 수정하지 못하도록 Firestore 규칙 적용
  • 재화 지급은 Cloud Functions 서버 검증 후 처리
  • 결제 도입 시 웹훅 검증 및 중복 지급 방지 이벤트 기록 사용
